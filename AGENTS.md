# Codex Agent Instructions

## Environment
1. Run `conda init` for your shell if it has not already been done.
2. Activate the project environment before any commands:
   `conda activate mut-epi-origin`

## Language
- Use British English in all comments, documentation and outputs.

## Coding Style
- Follow PEP 8.
- Do not code defensively.
- Add a module docstring at the top of each script, and update it when the script changes.
